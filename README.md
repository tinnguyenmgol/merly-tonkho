# Merly Ton Kho App

Deploy miễn phí bằng Streamlit Community Cloud.

## Files
- `app.py`
- `requirements.txt`
- `Logo Merly.jpg`

## Deploy
1. Tạo GitHub repo mới
2. Upload 3 file trên vào repo
3. Đăng nhập Streamlit Community Cloud
4. Chọn repo -> app.py -> Deploy

## Ghi chú
- App đọc các cột chuẩn:
  - Tên hàng hóa
  - Mã hàng hóa
  - Số lượng
  - Giá bán
  - Nhóm hàng
